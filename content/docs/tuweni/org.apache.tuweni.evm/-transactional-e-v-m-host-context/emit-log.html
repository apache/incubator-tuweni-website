emitLog

fun emitLog(address: ByteArray, data: ByteArray, dataSize: Int, topics: Array<ByteArray>, topicCount: Int): Unit (source)

Log function.

This function is used by an EVM to inform about a LOG that happened during an EVM bytecode execution.

Parameters

address - The address of the contract that generated the log.

data - The unindexed data attached to the log.

dataSize - The length of the data.

topics - The the array of topics attached to the log.

topicCount - The number of the topics. Valid values are between 0 and 4 inclusively.
