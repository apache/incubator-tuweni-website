RPCAsyncRequest(function: RPCFunction!, arguments: MutableList<Any!>!)

Parameters

function - RPCFunction!: the function to be in invoked. If the function is in a namespace, the first n-1 items in the array are the namespace followed by the function name (e.g. 'blobs.get' becomes ['blobs', 'get']).

arguments - MutableList<Any!>!: The arguments passed to the function being invoked. Each item can be any arbitrary object which is JSON serializable (e.g. String, Int, list, object.)
