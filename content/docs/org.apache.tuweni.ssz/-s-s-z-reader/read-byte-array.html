readByteArray

open fun readByteArray(): ByteArray! (source)

Read a byte array from the SSZ source. Note: prefer to use #readByteArray(int) instead, especially when reading untrusted data.

Exceptions

InvalidSSZTypeException - If the next SSZ value is not a byte array, or is too large (greater than 2^32 bytes).

EndOfSSZException - If there are no more SSZ values to read.

Return
ByteArray!: The byte array for the next value.
